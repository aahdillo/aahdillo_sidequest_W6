## Description

> This project lets users move a fox throughout a game by using keyboard inputs (left, right, and up arrow keys + space bar to attack).  
> The player must navigate through obstacles that appear along the path and collect the leaves.

---

## Interaction Instructions

Describe how to use or play your project.

The user start the game as soon as it loads. They use the left and right arrow keys to move horizontally and press the upper arrow to just and the space bar to attack. The goal of the game is to collect the leafes and without being killed by the fire or boars.
